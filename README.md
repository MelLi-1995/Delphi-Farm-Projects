# Delphi-Farm-Projects
# Ферма "Ново-Простоквашино" - Delphi проекты

Консольные приложения на Delphi с элементами ООП для RAD Studio.

## Проекты

### 1. Инкапсуляция (`1-encapsulation/`)
- Демонстрация private/public полей
- Get/Set методы
- Защита данных на уровне объекта

### 2. Иерархия фермы (`2-farm-hierarchy/`)
- Наследование таксономических классов
- Агрегация объектов
- Классы: Тип, Класс, Отряд, Семейство, Род, Вид

### 4. Паттерн Мост (`4-bridge-pattern/`)
- Разделение абстракции и реализации
- Интерфейсы для разных способов кормления
- Паттерн проектирования Bridge

## Запуск для разработчиков

Требуется: **RAD Studio** или **Delphi**

1. Откройте файл `.dpr` в RAD Studio
2. Нажмите `F9` для компиляции и запуска
3. Результат увидите в консоли
